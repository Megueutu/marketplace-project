# e-MAG

O **e-MAG** serve como um guia para desenvolvedores fazerem sites que qualquer pessoa consiga usar — mesmo quem:

- usa leitor de tela (deficiência visual)
- não consegue usar mouse (deficiência motora)
- tem dificuldades cognitivas

## Exigências do e-MAG

- **Imagens com descrição (alt text)**  
  → Para leitores de tela conseguirem “ler” a imagem  

- **Contraste de cores adequado**  
  → Para quem tem baixa visão conseguir enxergar  

- **Navegação por teclado**  
  → Para quem não usa mouse  

- **Vídeos com legenda**  
  → Para pessoas surdas  

## Origem

O e-MAG é baseado em um padrão internacional chamado  
**W3C (World Wide Web Consortium)**, que criou as diretrizes de acessibilidade da web (**WCAG**).

## Pra que serve?

- Tornar serviços públicos inclusivos  
- Garantir acesso igual à informação  
- Evitar que sites do governo excluam pessoas  

## ⚠️ Importante

No Brasil, órgãos públicos devem seguir o e-MAG, então ele não é só “boa prática” — é quase uma obrigação.
